# 4️⃣ 整理模式页（核心页面）

```
┌───────────────────────────┐
│                           │
│                           │
│        [ 当前照片 ]        │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
└───────────────────────────┘
```

核心操作：

```
← 左滑    下一张
→ 右滑    上一张

↑ 上滑    加入回收站
↓ 下滑    加入待定池

双击      撤销操作

双指↑     回收站
双指↓     待定池
```

浏览机制：

```
最后一张 → 左滑 → 第一张
```

无限循环。

# 5️⃣ 待定池页

```
┌───────────────────────────┐
│         待定照片           │
│                           │
│                           │
│        [ 照片 ]           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│                           │
│         ← 返回            │
└───────────────────────────┘
```

操作：

```
← 左滑   下一张
→ 右滑   上一张

↑ 上滑   移入回收站
↓ 下滑   保留
```

返回：

```
返回 → 整理模式页
```

# 6️⃣ 回收站页

```
┌───────────────────────────┐
│          回收站            │
│                           │
│     已删除 32 张照片        │
│     释放 480 MB           │
│                           │
│   ┌───────────────────┐   │
│   │       照片1        │   │
│   └───────────────────┘   │
│                           │
│   ┌───────────────────┐   │
│   │       照片2        │   │
│   └───────────────────┘   │
│                           │
│ [恢复]   [删除]   [全部删除] │
│                           │
│          ← 返回           │
└───────────────────────────┘
```

功能：

恢复：

```
恢复单张
恢复多张
恢复全部
```

删除：

```
删除单张
删除多张
全部删除
```

删除逻辑：

```
只有在回收站删除
才会真正删除系统照片
```

返回：

```
返回 → 整理模式页
```
